# 蓉e行交通众治平台

```bash
------------rex-platform
----------------rex-component           # 通用组件
----------------rex-mall                # 商城
----------------rex-message             # 消息模块
----------------rex-pay                 # 支付模块
----------------rex-starter-all         # 主要功能启动器
----------------rex-starter-icbc        # 工商银行接口启动器
----------------rex-starter-tmb-api     # 交管局接口启动器
----------------rex-starter-video       # 视频模块启动器
----------------rex-tmb                 # 交管局接口
----------------rex-ui                  # 后台管理前端页面
----------------rex-user                # 用户模块
----------------rex-video               # 视频模块，视频转码，截图
----------------rex-wechatr             # 微信
----------------rex-workflow            # 工作流，业务流程模块

```

# 启动
main方法运行: `rex-starter-all`模块下的 `RexAllApplication`
